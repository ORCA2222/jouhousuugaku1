  • 分配律と吸収律の証明
  • 分配律について

    分配律には二つの形があります。一つ目はA ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)、二つ目はA ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)です。これらを証明するためには、それぞれの等式の左辺と右辺が同じ集合を表していることを示す必要があります。これは、左辺の任意の元が右辺の集合に含まれ、かつ、右辺の任意の元が左辺の集合に含まれることを示すことにより達成されます。一つ目の法則A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)の証明を考えてみましょう。左辺の元はAに属するか、もしくはBとCの共通部分に属する元です。これは、右辺の集合(A ∪ B)と(A ∪ C)の共通部分、つまりAに属するか、BとCの少なくとも一方に属する元と一致します。よって左辺 = 右辺となります。同様に二つ目の法則A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)の証明も考えてみましょう。左辺の元はAと、BまたはCの少なくとも一方に属する元です。これは、右辺の集合(A ∩ B)または(A ∩ C)に含まれる元、つまりAとBまたはAとCの共通部分と一致します。よって左辺 = 右辺となります。以上より、分配律が証明されました。

  • 吸収律について
    • 吸収律について

      吸収律には二つあります。一つ目はA ∪ (A ∩ B) = A、二つ目はA ∩ (A ∪ B) = Aです。これらを証明するためには、それぞれの等式の左辺と右辺が同じ集合を表していることを示す必要があります。これは、左辺の任意の元が右辺の集合に含まれ、かつ、右辺の任意の元が左辺の集合に含まれることを示すことにより達成されます。

      一つ目の法則A ∪ (A ∩ B) = Aの証明を考えてみましょう。左辺の元はAに属するか、AとBの共通部分に属する元です。これらはすべて集合Aに含まれることは明らかです。したがって、左辺の任意の元は右辺の集合Aに含まれます。逆に、右辺の任意の元は左辺の集合A ∪ (A ∩ B)に必ず含まれます。したがって、左辺=右辺となります。

      同様に二つ目の法則A ∩ (A ∪ B) = Aの証明も考えてみましょう。左辺の元はAと、AまたはBの少なくとも一方に属する元です。これは、右辺の集合Aの定義と一致します。したがって、左辺の任意の元は右辺の集合Aに含まれます。逆に、右辺の任意の元は左辺の集合A ∩ (A ∪ B)に必ず含まれます。したがって、左辺=右辺となります。

      以上より、吸収律が証明されました。

  • ドモルガンの法則について

    まず、デ・モルガンの法則が二つあります。一つ目は(A ∪ B)^c = A^c ∩ B^c、二つ目は(A ∩ B)^c = A^c ∪ B^cです。これらを証明するためには、それぞれの等式の左辺と右辺が同じ集合を表していることを示す必要があります。これは、左辺の任意の元が右辺の集合に含まれ、かつ、右辺の任意の元が左辺の集合に含まれることを示すことにより達成されます。一つ目の法則(A ∪ B)^c = A^c ∩ B^cの証明を考えてみましょう。左辺の元はAでもBでもない元です。これは、右辺の集合A^cとB^cの共通部分、つまりAでもBでもない元と一致します。よって左辺=右辺となります。同様に二つ目の法則(A ∩ B)^c = A^c ∪ B^cの証明も考えてみましょう。左辺の元はAとBの共通部分ではない元です。これは、右辺の集合A^cまたはB^cのいずれかに含まれる元、つまりAとBの共通部分ではない元と一致します。よって左辺=右辺となります。以上より、デ・モルガンの法則が証明されました。

演習

Uを全体集合、Aをその任意の部分集合とします。

全体集合Uの定義から、全ての元xについてx∈Uが成り立つことが明らかです。すなわち、任意の元xについて、xがAに属するならばxはUにも属すると言えます。

A⋂Uの定義から、A⋂UはAとUの共通部分、つまりAにもUにも属する全ての元からなる集合です。前述の議論から、Aに属する全ての元はUにも属すると言えるので、A⋂UはAの全ての元を含むことがわかります。つまり、A⋂U⊇Aが成り立ちます。

逆に、A⋂Uの任意の元xは、定義から明らかにAに属します。従って、A⋂U⊆Aも成り立ちます。

以上から、A⋂U⊆AかつA⋂U⊇Aが成り立つので、A⋂U=Aが証明されました。
